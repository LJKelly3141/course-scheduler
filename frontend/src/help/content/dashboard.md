# Dashboard

The Dashboard provides a real-time overview of your selected term's schedule, highlighting key metrics, conflicts, and potential issues at a glance.

## Getting There

Click **Dashboard** in the sidebar, or navigate to the app — it's the default landing page.

## Key Concepts

- **SCH** — Student Credit Hours (enrollment x credits per section)
- **SCH/FTE** — Student Credit Hours per Full-Time Equivalent instructor, a productivity metric
- **Hard Conflict** — A scheduling error that must be fixed before finalizing
- **Soft Warning** — An advisory issue you can dismiss if it's intentional

## Summary Cards

Six cards across the top show at-a-glance metrics for the selected term:

| Card | What It Shows |
|------|--------------|
| Sections | Total number of sections in the term |
| Credits Offered | Sum of credits across all sections |
| Instructors | Number of instructors with assigned sections |
| Projected SCH | Estimated student credit hours based on enrollment caps |
| SCH/FTE | Productivity ratio |
| Conflicts | Count of unresolved hard conflicts |

## Conflicts & Warnings Panel

This panel groups issues into sequential sections:

- **Hard Conflicts** — Must be resolved before the term can be finalized. Click a conflict to see which meetings are involved.
- **Soft Warnings** — Advisory issues like credit overloads or room capacity mismatches. Click the **dismiss** button to hide warnings you've reviewed.
- **Prerequisite Warnings** — Highlights courses with prerequisite scheduling issues, such as a prerequisite course not being offered in a prior term.
- **Dismissed** — Previously dismissed warnings. Click **restore** to bring them back.

## Missing Courses

Shows courses that were offered in previous terms of the same type (e.g., previous Fall terms) but don't have any sections in the current term. This helps you catch courses that may have been accidentally omitted.

## Course Analytics Table

A table listing all courses in the term with:
- Projected enrollment and trends (growing/declining indicators)
- Number of sections and total capacity
- Fill rate projections

## Academic Year Display

The current academic year is shown at the top of the dashboard. Terms are automatically assigned to academic years based on the start month configured in Settings.

## Tips

> **Tip:** Select different terms using the **term dropdown** in the sidebar header to quickly compare term overviews.
