Key Design Decisions

  • Roster panel — Always visible, shows name + type + quick workload status. Filter by type. Search. Inactive instructors shown in italic.
  • Detail panel — 4 tabs: Profile, Availability, Workload, Notes. Term selector in the header affects Availability + Workload tabs.
  • Profile tab — 3 logical sections: Contact, Employment, Scheduling Preferences. All editable inline.
  • "+ New Instructor" button at bottom of roster opens a guided wizard (next mockup).
